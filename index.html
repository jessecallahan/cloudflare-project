Description

Here is my interpretation of the Cloudflare A/B Full Stack testing problem. This application takes two url links nested in an api and parses the values randomly at the click of a button. I am in the middle of a JS/React bootcamp that will be finished June 2020 (working remotely right now). I wish I could have built this application with React or any other React style library/framework however I am just not there in my training yet. Obviously, these objects aren't immutable and wouldn't work in a React App. That is the true goal of my internship though, mastering state and mutation. If given a chance, I'd like to resubmit this project as a React App when I'm finishing up my program in June.
